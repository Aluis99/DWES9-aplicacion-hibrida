## Descripción

Aplicación desarrollada para la tarea DWES9, correspondiente al desarrollo de aplicaciones web híbridas.

La aplicación consume un servicio web externo en formato JSON utilizando PHP y `file_get_contents()`.

## Apartados realizados

### RA9_d

El código fuente se publica en un repositorio GitHub.

### RA9_f

Se consume una API REST pública, JSONPlaceholder, y se muestran publicaciones en una página web.

### RA9_h

Se desarrolla una página de prueba con navegación, diseño visual, comentarios PHPDoc y prueba de carga con JMeter.

## Archivos principales

- `index.php`: página principal de la aplicación.
- `servicio.php`: página que consume la API externa.
- `funciones.php`: funciones PHP documentadas con PHPDoc.
